#  Carbon Shadow Twin + Human Analyzer

A futuristic full-stack web/mobile application that visualizes a user’s carbon footprint as a living digital avatar (“Shadow Twin”) and provides intelligent insights using AI.

---

##  Tech Stack

* **Frontend:** React (Vite), Tailwind CSS, Framer Motion
* **Backend:** Python (FastAPI) / Node.js (optional)
* **Database:** Firebase / Google Sheets (No-code option)
* **AI Integration:** OpenAI API (Chatbot)
* **Voice:** Web Speech API / Voiceflow
* **Visualization:** Custom UI + Avatar Logic Engine

---

##  Live Demo

[Click here to view the app](https://carbon-shadow-twin.netlify.app)

---

## features Implemented

* ** Shadow Twin Dashboard:**
  A dynamic avatar that changes appearance based on carbon footprint (clean → polluted).

* ** Carbon Tracker:**
  Tracks daily activities like travel, food habits, and electricity usage.

* ** Human Analyzer:**
  AI-based insights that detect behavior patterns and suggest improvements.

* ** AI Chat Assistant (Eco Coach):**
  Helps users reduce carbon footprint with smart suggestions.

* ** Voice Input Feature:**
  Users can interact with the system using voice commands.

* ** Activity Logger:**
  Records daily habits and calculates emissions.

* ** Gamification System:**
  Users level up by improving sustainability score.

* **Responsive UI:**
  Clean, modern UI with glassmorphism and smooth animations.

## Core Concept

This project introduces a new way to understand carbon footprint by turning it into a **visual and emotional experience**.

Instead of numbers:
➡️Users see their impact as a **living shadow**
➡️Behavior directly changes its appearance
➡️AI helps improve lifestyle
